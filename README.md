# MythWare Tool v5.0.0

用于 Windows 进程管理与极域电子教室辅助的小工具，个人学习练手项目。



## 项目介绍

本项目从 C 语言控制台版本开始，逐步迭代到 Qt 图形界面，全程为学习开发所用。

当前 v5.0.0 采用纯代码手写界面，功能已测试稳定。



## 主要功能

- 进程挂起、恢复、结束

- 重启学生端程序

- 屏幕广播解锁

- 配置文件路径管理

- 自定义消息提示框

- 多页面切换界面



## 使用说明

- 首次运行自动生成配置文件

- 可在设置中修改学生端路径

- 路径只需要填写程序所在文件夹，不需要包含文件名
